# Ultrasonic-Piano
This is a piano created mainly with pic18f4520 and ym2413 (um3567)

## Environment
- MPLAB X IDE
- XC8 Compiler
- PicKit 3

## Hardware
- Microchip PIC18F4520 microcontroller
- HC-SR04 sensor
- YM2413 (UM3567)
- 7-segment-display
- A self-made wood piano
